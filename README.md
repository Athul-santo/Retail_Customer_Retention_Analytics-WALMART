Retail Customer Retention Analytics – Walmart

Project Overview:
Interactive Power BI dashboard analyzing Walmart customer behavior by consolidating demographics, purchase history, store/e-commerce performance, and loyalty data. Segments high-value, repeat, and churned customers to provide actionable insights for improving retention, loyalty, and regional strategies.

Key Features:

Churn & Retention Metrics by region, channel, income, and loyalty tier
Repeat Purchase Analysis and product category insights
Promotion & Loyalty Impact evaluation
Store & Channel Performance analysis
Customer Lifetime Value (CLV) segmentation

Technologies Used:
Power BI, DAX, Power Query, CSV data
